# 🔥 哪吒·魔童降世 — 3D 桌面精灵外观设计规范

> 参考来源：《哪吒之魔童降世》(2019)、《哪吒之魔童闘海》(2025)  
> 美术风格：中式水墨 × 赛博朋克融合，Q版化适配桌面精灵

| 要素 | 设定 |

## 二、面部设计

### 2.1 脸部
- **脸型**：瓜子脸偏圆，带婴儿肥（Q版化但保留棱角）
- **肤色**：暖白偏小麦（中式少年肤色）
- **眉毛**：剑眉上挑，眉头略粗、眉尾锋利
- **眼睛**：
  - 大眼凤眼，眼角上翘
  - 虹膜：黑褐色 → **燃状态变金色/红色（Shader 控制）**
  - 烟熏妆/黑眼圈（魔丸印记）：上下眼睑淡淡烟熏，右眼下方略重
- **鼻子**：小巧挺拔
- **嘴**：嘴角微翘（标志性邪笑），Q版可做夸张表情

### 2.2 发型
- **款式**：传统丸子头 × 2（左右对称双髻）
- **发色**：深棕黑，发尾带深红色渐变
- **刘海**：中分两缕垂至眉骨，后发披散至肩
- **发饰**：红绳束发，两侧各一根飘带垂下

### 2.3 BlendShape / 表情混合形状
```
happy    → 嘴角上扬 + 眼睛弯月
sad      → 嘴角下垂 + 眉毛八字
angry    → 眉头紧锁 + 咬牙切齿
surprised→ 眼睛瞪大 + 嘴巴圆形
fired_up → 火焰瞳孔 + 嘴角邪笑 + 额头青筋（可选）
```

## 三、服装与装备

### 3.1 上衣
- **款式**：红色无袖马甲，敞怀穿
- **材质**：粗麻布质感 + 破损边缘
- **纹样**：胸前莲花暗纹（淡金色刺绣）
- **内搭**：深棕色缠腰布

### 3.2 下装
- **裤子**：深褐色七分灯笼裤
- **腰带**：粗麻绳束腰，飘带垂至膝

### 3.3 四大法器

| 法器 | 位置 | 设计 | 动态效果 |
|------|------|------|----------|
| 🔥 **混天绫** | 绕双臂/绕颈飘飞 | 半透明红色绸缎，约身长2倍 | 常驻飘动动画，愤怒时火焰化 |
| ⭕ **乾坤圈** | 右手腕（缩小态） | 金色金属环，表面刻云纹 | 战斗时变大飞出 |
| 🎡 **风火轮** | 双脚 | 双轮盘，青色火焰喷射 | 常驻旋转+粒子火焰 |
| 🗡️ **火尖枪** | 右手持（或背后） | 红缨长枪，枪尖三叉 | 战斗时挥动特效 |

### 3.4 火焰特效
- **颜色**：赤红 → 橙 → 金，多层粒子叠加
- **触发**：`fired_up` 情绪时全身火焰包围
- **Shader**：Panner + Dissolve + Emission，URP/内置管线双兼容

## 五、动画状态机 (Animator)

### 5.1 状态列表

| 状态名 | 触发条件 | 时长 | 描述 |
|--------|----------|------|------|
| `idle` | 默认 | 循环 | 双手插兜，微微晃动，混天绫飘动 |
| `bounce` | happy | 1.5s | 原地蹦跳，双手挥动，风火轮喷火星 |
| `spin` | excited | 2s | 踩着风火轮旋转一圈，混天绫环绕 |
| `wave` | 打招呼 | 2s | 单手举起挥动（乾坤圈那手），嘴角邪笑 |
| `yawn` | sleepy | 3s | 打哈欠，揉眼睛，混天绫垂地 |
| `jump` | surprised | 1s | 向后小跳，眼睛瞪大，双手防御姿态 |
| `stomp` | angry | 2s | 跺脚，风火轮喷火，拳头握紧 |
| `droop` | sad | 循环 | 低头垂肩，混天绫无精打采垂下 |
| `hide` | shy | 2s | 转身背对，用混天绫挡住半张脸 |
| `battle_pose` | fired_up | 循环 | 火尖枪指前，混天绫火焰化，全身燃烧粒子 |
| `stretch` | 空闲随机 | 3s | 伸懒腰，风火轮离地悬空 |
| `look_around` | 空闲随机 | 2s | 左右张望，手搭凉棚 |

### 5.2 过渡规则
- **Any State → battle_pose**：当 `fired_up` 情绪值 > 0.6 时，0.3s 过渡
- **battle_pose → idle**：`fired_up` < 0.2 且持续 3s 后，带火焰消散特效过渡

## 六、技术实现指南

### 6.1 建模建议
- **工具**：Blender (免费) 或 VRoid Studio → 导入 Unity
- **VRM 格式**优先（自带表情混合形状和骨骼）
- 如果没有 VRM，用 FBX + 手动配置 Humanoid Rig

### 6.2 Shader 要点
- **皮肤**：Toon Shader with Ramp Texture
- **混天绫**：Transparent + Vertex Animation (飘动)
- **火焰**：Particle System (GPU Instancing) + Additive Blend
- **眼瞳变色**：Material Property Block 运行时切换

### 6.3 性能预算（手机端）
- 三角面：≤ 12000
- Draw Call：≤ 8（不含 UI）
- 粒子数：≤ 200
- 贴图：512×512 为主，1024×1024 仅面部

### 6.4 可获取的免费资源参考
- [VRoid Hub](https://hub.vroid.com) — 搜索 "哪吒" 或 "少年" 找基底模型
- [Unity Asset Store](https://assetstore.unity.com) — Toon Shader、火焰粒子
- [Mixamo](https://www.mixamo.com) — 骨骼动画（idle/walk 等）
- [Sketchfab](https://sketchfab.com) — 搜索 "nezha" 3D 模型（CC 授权）
