## 五、部署前建议

### 必须配置
```bash
# 管理端点认证密钥
export ADMIN_API_KEY="$(openssl rand -hex 32)"

# Grafana 管理员密码
export GRAFANA_ADMIN_PASSWORD="$(openssl rand -hex 16)"
```

### 推荐架构

```
Internet → TLS反向代理(nginx/Caddy) → Gateway(127.0.0.1:8800)
                                       ├── WAF 扫描
                                       ├── 限流 (TokenBucket)
                                       └── 认证 (ADMIN_API_KEY)
```
